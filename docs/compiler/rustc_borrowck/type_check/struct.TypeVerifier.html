struct TypeVerifier<'a, 'b, 'tcx> {
    cx: &'a mut TypeChecker<'b, 'tcx>,
    promoted: &'b IndexSlice<Promoted, Body<'tcx>>,
    last_span: Span,
}
Expand description

Verifies that MIR types are sane to not crash further checks.

The sanitize_XYZ methods here take an MIR object and compute its type, calling span_mirbug and returning an error type if there is a problem.
