    A flag that allows to distinguish PTHREAD_MUTEX_NORMAL from PTHREAD_MUTEX_DEFAULT. Since in glibc they have the same numeric values, but different behaviour, we need a way to distinguish them. We do this by setting this bit flag to the PTHREAD_MUTEX_NORMAL mutexes. See the comment in pthread_mutexattr_settype function.
