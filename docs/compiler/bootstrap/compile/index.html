Implementation of compiling various phases of the compiler and standard library.

This module contains some of the real meat in the rustbuild build system which is where Cargo is used to compile the standard library, libtest, and the compiler. This module is also responsible for assembling the sysroot as it goes along from the output of the previous stage.
